# House Price Prediction using Machine Learning

## Project Overview
This project predicts house prices based on property features such as area, bedrooms, bathrooms, parking, and furnishing status.

## ML Type
Supervised Learning

## Task
Regression

## Goal
The goal is to build a beginner-friendly end-to-end machine learning project and understand the complete ML workflow from data loading to model evaluation.

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook
